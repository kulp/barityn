```
tyrga 0.1.0
Darren Kulp <darren@kulp.ch>
tyrga translates JVM bytecode (.class files) into tenyr source code.

USAGE:
    tyrga [SUBCOMMAND]

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

SUBCOMMANDS:
    demangle     Decodes mangled tenyr symbols into strings
    help         Prints this message or the help of the given subcommand(s)
    mangle       Mangles strings of bytes into valid tenyr symbols
    translate    Translates JVM .class files into tenyr .tas assembly files
```
